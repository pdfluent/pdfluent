# Pdfium Native Library

This directory holds the pre-built Pdfium binary required at runtime.

## Setup

Run from the project root:

```bash
./scripts/setup-pdfium.sh
```

This downloads the correct Pdfium binary for your platform into `lib/`.

## Source

Binaries from [nicbarker/pdfium-binaries](https://github.com/nicbarker/pdfium-binaries) (Apache-2.0).
